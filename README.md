# GoTravel Website

Site web de voyage moderne développé en HTML, SCSS et méthodologie BEM, avec une structure propre, responsive et optimisée.

## Pages
- Accueil
- Voyages
- Activités
- Contact
- Connexion
- Inscription
- 6 pages destinations : Paris, Bali, Santorin, Tokyo, New York, Dubaï

## Technologies
- HTML5
- SCSS
- CSS
- BEM
- Responsive design
- Menu burger sans JavaScript

## Utilisation
Ouvre `Accueil.html` dans ton navigateur.
Si tu modifies `style.scss`, active Watch Sass dans VS Code pour générer `style.css`.
